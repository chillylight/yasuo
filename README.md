# 图片压缩网站

这是一个简洁优雅的图片压缩工具网站，采用苹果设计风格，提供以下核心功能：

## 功能特点

1. 支持上传 PNG、JPG 等格式图片
2. 提供图片压缩功能，可自定义压缩比例
3. 实时预览原图和压缩后的效果
4. 显示压缩前后的文件大小对比
5. 支持压缩后图片下载

## 技术实现

- 使用原生 HTML5、CSS3 和 JavaScript 开发
- 采用 FileReader API 实现图片预览
- 使用 Canvas API 进行图片压缩
- 响应式设计，适配各种设备
- 模仿苹果设计风格，注重细节体验
